🔍 What I did:
• Built a Decision Tree Classifier on the Bank Marketing Dataset
• Encoded categorical variables using Label Encoding
• Split data 80% training / 20% testing
• Achieved 78% Model Accuracy!
• Visualized the actual Decision Tree structure

💡 Key Insights:
✅ Call Duration was the #1 predictor of purchase (53% importance!)
✅ Account Balance was the 2nd most important feature
✅ Previous campaign outcome strongly influenced purchase decision
✅ Confusion Matrix showed strong performance on non-purchase prediction

🛠️ Tools Used:
• Python, Pandas, NumPy
• Scikit-learn — DecisionTreeClassifier, train_test_split, metrics
• Matplotlib, Seaborn
